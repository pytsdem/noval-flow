# Inputs And Outputs

这个目录汇总各个 Agent / 功能的输入输出，方便你调试和调整协议。

## 目录说明

- `agent_capabilities.md`
  - 每个 Agent 的能力、调试入口和主要输入输出
- `writer_agent.md`
  - WriterAgent 四种模式的输入输出
- `research_agent.md`
  - ResearchAgent 的输入输出
- `critic_agent.md`
  - CriticAgent 的输入输出
- `master_agent.md`
  - MasterAgent pipeline 的输入输出
- `schemas_index.md`
  - 关键 schema 的索引
- `examples/`
  - 结构化 JSON 示例，方便你直观看协议
